### Project Overview

 In this loan approval project on the basis of various columns you  will be able to decide whether we had to provide loan to that customer or not on their historiacal database and  if not then which type of offer we can offer to him.


### Learnings from the project

 You will able to learn pandas dataframe  downloading dataset,concepts of groupby,pivot ,apply aggregations methods,slicing  and making subset of given dataset and perform different mathematical operations.


### Approach taken to solve the problem

 The whole project is categorised into 6 tasks startinf from loading dataset  to cleaning of data and finding the suitable answers of questions by using pandas dataframe and its attributes or properties.


### Challenges faced

 It is very easy project for beginners.
